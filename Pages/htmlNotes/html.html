INPUTS in HTML

    • The input tag specifies an input field where the user can enter data.
    • The input element is the most important form element.

<input type="text">


Attributes in input tag:

  • type:

    Type property defines the type of input control for single line input types
    must be set to text.

  • Name:

    Name attribute is used to give names to the controls so it can be recognise bye server.

  • value:

    This attribute is used to provide initial value inside the control.

  • size:

    It specifies width of contorl.

  • min/max length:

    It defines the range of character that a control can have.

  • placeholder

    It use to give a message to user about the type of input or other. <input type="text" placeholder="enter your name">

HTML FORM

HTML form are required when we want to make our webpage dynamic with the help of form we can collect user data
and send and recive information from the server. forms can be used like registration, login etc. To make it dyanamic
we require server scriting languages such as ASP, PHP and USP.

Form syntex

Basic attribute of form tag:-

  1. Action:

    This attribute process backend script written in ASP, PHP, USP and action method is also used
    to send user information to the server

  2. Method:

    Method attribute used to upload user data Get/Post method are used to send teh information.

  3. Target:

    This attribute specifies teh target windows or frame where the result of script will be displayed
    Target attribute can be _blank,self

  4. enctype:

    enctype attribute defines how the browser will encode the data before if is sent to the user This attribute can urlencoded
    have two values:
    • application/X-www-form: it is the most simple method to encoded data. It is used to encode simple text information
    • multiport/form-data: This is used to upload binary information, such as image, word,file etc.

Note:

both get and post method is used to transfer data from client to server, In HTTP protocol but main difference between post & get
method is that get carries request paraeter appended in URL string while post carries request parameter in message body.
